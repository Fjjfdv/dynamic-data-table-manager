# Dynamic Data Table Manager

A dynamic and customizable data table built using **Next.js**, **Redux Toolkit**, and **Material UI**.

### Features:
- Show/Hide Columns
- Add Custom Fields (like Department, Location)
- CSV Import/Export
- Persistent Column State with LocalStorage
